# bussler.github.io
Repository for personal Vuejs-based [website](https://bussler.github.io/) with github pages.  
The website is used for showing personal projects that amassed over the time.

## Getting started

### Install

`npm install`

### Format

`npm run format`

### Build

`npm run build`

### Serve

`npm run dev`

## Docker

`docker compose up -d --build`
